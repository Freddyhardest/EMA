O.U. Walking Club: Membership

Trial web page for the Open University Walking Club.

This form allows the user to apply for membership to the O.U. Walking Club. The user can enter their full name, home address, preferred payment method, date of birth, email, phone number and alternative contact number. The user can also indicate whether they consent to receive announcement emails by entering Y or N in the appropriate field. The data entered by the user can be saved to a file by clicking the "Save data to file" button.
